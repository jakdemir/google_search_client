# Google Search Client

A Java client for Google Search API with JSON response deserialization.

## Overview

Sample implementation of a Google Search API client demonstrating:
- HTTP request handling
- JSON parsing and deserialization
- Object mapping from API responses

## Technologies

- Java
- JSON parsing libraries
- Google Search API

## Features

- Send search queries to Google
- Parse JSON responses
- Map results to Java objects

## Usage

```java
GoogleSearchClient client = new GoogleSearchClient();
SearchResults results = client.search("your query");
```
